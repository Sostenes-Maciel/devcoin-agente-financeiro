# Base de Conhecimento

O agente utiliza os arquivos na pasta `/data` para fundamentar suas respostas:
- **transacoes.csv:** Analisa o fluxo de caixa para sugerir cortes.
- **perfil_investidor.json:** Entende as metas (ex: reserva de emergência e hardware).
- **produtos_financeiros.json:** Sugere onde alocar a economia para render até a data da compra do equipamento.
